# 📊 Power BI Data Analysis Project – Sales Dashboard

## 📘 Overview
This project was completed during a **Data Technician Bootcamp**, focusing on **data analysis, data transformation, and business intelligence (BI)** using Power BI.

Using a retail and sales dataset, the project demonstrates how to clean, transform, and model data to build **interactive reports and dashboards** that deliver actionable business insights.

---

## 🧰 Skills & Tools Used

### 🔧 Power BI & Data Modelling
- Developed **interactive Power BI reports and dashboards**
- Used **Power Query** for data cleaning and transformation (ETL process)
- Built **data models** by defining relationships between tables
- Applied **best practices for data organisation and reporting**

---

### 🔍 Data Transformation (Power Query)
- Cleaned and formatted raw sales data
- Handled missing values and inconsistent data types
- Performed data transformations such as:
  - Filtering rows and columns
  - Removing duplicates
  - Creating new derived columns
- Prepared data for efficient analysis and reporting

### 📊 DAX (Data Analysis Expressions)
- Created **calculated columns and measures** for key metrics:
  - Total Sales
  - Profit
  - Sales Growth
  - Average Order Value
- Used DAX functions to perform:
  - Aggregations
  - Conditional logic
  - Time-based calculations

## 📈 Data Visualisations

A range of visualisations were built to effectively communicate insights:

- 📊 **Bar Charts** – Sales and profit comparison across products and categories  
- 📉 **Line Charts** – Sales trends over time (monthly/yearly)  
- 🥧 **Pie Charts** – Sales distribution by category or region  
- 🗺️ **Map Visualisations** – Geographic analysis of sales performance  
- 📌 **Cards/KPIs** – High-level metrics (Total Sales, Profit, Orders)

## 🔍 Key Analysis

### 📌 Sales Performance
- Identified **top-performing products and categories**
- Analysed **revenue and profit trends over time**
- Measured **overall business performance using KPIs**

### 📌 Customer & Regional Insights
- Compared **sales performance across regions**
- Highlighted areas with **high and low profitability**
- Analysed customer purchasing behaviour patterns

### 📌 Business Insights
- Identified opportunities to **improve sales and profitability**
- Detected **seasonal trends and demand patterns**
- Supported **data-driven decision-making**

---

## 🧠 Data Storytelling
- Designed dashboards with a **clear and logical narrative structure**
- Used **visual hierarchy, colour, and layout** to highlight key insights
- Enabled **interactive exploration** through slicers and filters (e.g., region, category, time)
- Tailored reports for **non-technical stakeholders and business users**
